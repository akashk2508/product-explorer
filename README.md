## 🛍️ Product Explorer

A responsive web application built using React that allows users to browse and search products fetched from a public API. This project demonstrates API integration, state management, and responsive UI design.


## 🚀 Features

* 📦 Fetch and display products from an external API (DummyJSON (https://dummyjson.com/products))
* 🔍 Real-time search functionality
* ⏳ Loading state while fetching data
* ❌ Error handling for failed API requests
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🧩 Component-based architecture


## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5 & CSS3
* Public API: DummyJSON (https://dummyjson.com/products)

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone (https://github.com/akashk2508/product-explorer)
```

2. Navigate to project folder

```
cd product-explorer
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```
